# C_to_Rust
Project to port a C application to Rust (tic-tac-toe game)


# run Rust   
```bash
cd Rust_tic_tac_toe/
cargo run
```

# run C 
windows
```bash
cd C_tic_tac_toe/
gcc tic_tac_toe.c -o tic_tac_toe
.\tic_tac_toe.exe
```

linux
```bash
cd C_tic_tac_toe/
gcc tic_tac_toe.c -o tic_tac_toe
.\tic_tac_toe
```
